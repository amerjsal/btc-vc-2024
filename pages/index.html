IT-Web Software Developer Associate of Applied Science Degree

Description:

This program offers and Associate's Degree and will teach you a little bit of everything when it comes to software development. You will write/design programs using quite a few programming languages to maintain websites, work with server spaces, learn GitHub, and more! This program is designed specifically to prepare you for an entry-level web development job.

Additional Information:

Number of Credits: 60 credits

Expected wage, job titles, outcomes:

  • Median Wage: $83,100
  • Web Designer, Web Development Specialist, Web Architect, e-Commerce Support Specialist, Programmer/Analyst
  • Bachelor's degree or career outcome

Estimated Tuition Cost: $9,606
