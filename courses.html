Course Descriptions:

Introduction to Business and Management

You’ll be learning basic business operations, and the basics of the activities, functions, and principles of businesses. You’ll learn the responsibilities and challenges of operating a business. It emphasizes human relations, management, marketing, finance, human resources, global business, and the process of starting a business.

Intro to Adobe Illustrator

This course prepares you to use Adobe's Illustrator software to create various digital art properly. It’ll cover the basics you’ll need to know of computer illustration and its use in print and web media. You’ll work with objects, the pen, brushes, text, color, special effects, symbols, and gradients as you create portfolio-quality art.

Intro to Photoshop- Designing and Editing

Introducing you to using Adobe Photoshop to edit, enhance, and combine digital images which you’ll prepare for printing and web use. You will learn photo correction, masks, channels, page backgrounds, brushes and optimizing images for web use.

Introduction to Programming

Specifically for students with very little or no prior programming experience, this course will introduce the basic concepts of object oriented programming. You’ll learn about integers, strings, variables, and objects. Learning to analyze the core elements of programs, run them at the command line, do some debugging, and create a very simple program all on your own.

Web Servers and Web Server Administration

In this course, you’ll prepare an environment, install, and configure an open source web server. Learning to configure your server and install services to use for projects throughout the remainder of the semester. You’ll also use DNS to connect a URL to the virtual server space you’ll have created.

Markup Basics 1

You’ll analyze the basics of web pages, which are markup languages. Analyzing how web browsers read markup language to display web pages. You’ll organize text and add images/videos to make useful and engaging web pages.

Design Basics for Front-end Development 1

This course will explore the key elements of effective design, while introducing you to the fundamentals of design and style coding. You’ll use scripting to enhance your projects with color, background images, and customized fonts.

Intro to Digital Photography & Color

This course will guide you through the basic photography principles and editing of files. You’ll learn specialized shooting and lighting techniques to enhance image quality. Additionally, you will develop skills in printing, cropping, retouching, and organizing their work by creating an effective file management system.

High Level Scripting

You will apply skills that include syntax, variables, conditionals and functions, along with a focus on scripting for the web with Javascript.

Design Basics for Front-end Development 2

This course will analyze design elements of web pages and evaluate more advanced level layout creation and customization with scripting.

Markup Basics 2

This course will focus on creating interactive web pages, showcasing data effectively, and coding to make content more accessible to everyone.

Relational Database Design & Development

This course gives you knowledge on database concepts for design and development. You will design and develop a database and write code to manipulate the data in the database.

Mathematical Reasoning OR Intermediate Algebra with Applications

Exploring graphs, proportional relationships, algebraic reasoning, and problem solving using linear, exponential and more mathematical models. You will learn properties of numbers, order of operations, algebraic solutions for linear equations and inequalities, operations with polynomial and rational expressions, operations with rational exponents and radicals, algebra of inverse, logarithmic and exponential functions.

Introduction to Sociology

Introduces you to concepts of sociology, including: culture, socialization, social stratification, multiculturalism, and the five institutions, including family, government, economics, religion, and education. You’ll also learn about demography, deviance, technology, environment, social issues, social change, social organization, and workplace issues.

Project & Framework Management

You’ll evaluate various tools in web development to plan, organize, and execute digital projects including storyboards, wireframes, and usability testing.

Version Control

You will explore the basics of software version control (SVC), including process management. You will learn a brief history of SVC and analyze the fundamental concepts of check-in/checkout, forking, merging, commits, and distribution. You will also create a repository and manage files within it.

Cybersecurity

Learning the importance of website and server security and data confidentiality through basic aspects of information security. Introduces you to common methods of securing data.

Content Management Systems 1

Learn the different options for content management systems (CMS), both proprietary and open source. You’ll evaluate the options for systems and determine the best solution for different scenarios.

Content Management Systems 2

You’ll install and configure an open source CMS, and user training/roll out

Search Engine Optimization

You’ll identify what search engine optimization (SEO) is and why it's a very important skill for front-end developers to have. You will analyze free and paid keyword search tools, identify keywords rank, and generate a keyword plan.

Front End Framework

You’ll develop skills and knowledge you need to code with a front-end framework.

Mobile Application Development

You will identify and apply how mobile technology works, tools for development, specifications, and implementation.

Emerging Technologies Research

You’ll learn to identify a problem that can be solved by technology and research the options for solving the problem using a programming language or web tool.

Speech

Learn how to give oral presentations to small and large groups. Topic selection, audience analysis, research, supporting evidence, delivery techniques, and more. Includes informative, persuasive, and occasion speech presentations.

Introduction to Psychology

This course teaches different aspects of human behavior. Including a survey of theoretical foundations in learning, motivation, emotions, personality, physiological factors, and social influences.

Responsive Design

You will analyze the mobile first responsive design method, outlining the basics of creating websites that are functional cross-platform and device.

Programming

You will identify and apply the foundations of programming on the LAMP stack (Linux, Apache, MySQL, and PHP), from an overview of a server-side programming language. You will learn to program with PHP and create functional web pages with server-side scripting.

Social Media

You will learn the uses of social media tools to manage accounts. Learning about the integration of social media with websites and the options in social media for advertising, including creating ads and determining audiences. You’ll also explore open Application Programming Interfaces (APIs) within software and build a program that connects to a social media platform.

Resume and Portfolio Development for Web Developers

In this course, you’ll review the work you’ve completed during your time in the program and develop a full portfolio website. From this website portfolio review, students will also learn resume writing and employment readiness skills.

Emerging Technologies Capstone

You will use the skills learned in the program to identify a digital solution to a complex problem. You will outline the problem, find a solution, and present your findings for peer review and collaboration.
